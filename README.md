# T20-Worldcup_2022-Dream-Team-Analysis-Dashboard
🏏 T20 World Cup 2022 Dream Team Analysis Dashboard

#Overview

This project analyzes player performances from the ICC Men's T20 World Cup 2022 using Power BI. The goal is to identify the best players for each role and build a data-driven Best Playing XI.

#Dashboards

1. Power Hitters / Openers

Analysis of aggressive opening batsmen based on Runs, Strike Rate, Boundary %, and Batting Average.

2. Anchors / Middle Order

Analysis of consistent middle-order batsmen who can stabilize the innings.

3. Finishers

Analysis of lower-order batsmen capable of accelerating the scoring rate in death overs.

4. Specialist Fast Bowlers

Evaluation of pace bowlers using Wickets, Economy Rate, Bowling Strike Rate, and Dot Ball Percentage.

5. All-Rounders

Analysis of players contributing with both bat and ball.

6. Final Playing XI

Selection of the strongest T20 World Cup 2022 team based on insights from all dashboards.

#Tools Used

- Power BI
- Power Query
- DAX
- Excel
- JSON
- Numpy

#Features

- Interactive dashboards
- Role-based player selection
- Player images integration
- Advanced performance metrics
- Data-driven final XI selection

Project Outcome

Created a complete cricket analytics solution that identifies the best T20 World Cup 2022 playing XI using statistical analysis and visualization techniques.
